# Advanced Problem Solving with TypeScript & OOP

This project contains solutions to various TypeScript problems and two blog posts on advanced TypeScript concepts. It demonstrates fundamental TypeScript concepts, including data typing, interfaces, generic functions, type checking, and Object-Oriented Programming (OOP) principles.

## 📁 File Structure

- **solutions.ts**: Contains all TypeScript coding solutions for Problems 1 to 7.
- **blog-1.md**: A blog post about TypeScript Generics and their role in building reusable, type-safe components.
- **blog-2.md**: A blog post discussing the four pillars of Object-Oriented Programming (OOP) and their importance in managing complexity.
- **README.md**: Overview of the folder content.

## 💻 Solved Problems

1. **Even Number Filter**: A function to filter even numbers from an array.
2. **String Reversal**: A function to reverse a given string.
3. **Type Guarding**: A function using type guards to identify string or number types.
4. **Property Getter**: A generic function to safely retrieve properties from objects.
5. **Book Status Toggler**: An interface-based function to add and toggle a `isRead` property on book objects.
6. **Class Inheritance**: Implementation of `Person` and `Student` classes with inheritance and detailed reporting.
7. **Array Intersection**: A function to find common elements between two numeric arrays.

## 📝 Blog Topics

1. **Generics**: Explains how Generics allow for reusable and strictly typed components.
2. **Four Pillars of OOP**: Discusses Inheritance, Polymorphism, Abstraction, and Encapsulation in large-scale projects.
